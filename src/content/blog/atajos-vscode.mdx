## Maximiza tu Productividad con Visual Studio Code ⚡

Visual Studio Code (VS Code) es uno de los editores más usados por desarrolladores de todo el mundo. Sin embargo, muchos no aprovechan todo su potencial. Aquí te traigo una lista de atajos de teclado y hacks esenciales que transformarán tu experiencia como programador. 🚀

---

## 🔥 Atajos de Teclado Esenciales en VS Code

### ✨ Navegación Rápida

- **Abrir un archivo rápidamente** → `Ctrl + P` (Windows/Linux) | `Cmd + P` (Mac)
- **Ir a una línea específica** → `Ctrl + G` | `Cmd + G`
- **Mostrar la paleta de comandos** → `Ctrl + Shift + P` | `Cmd + Shift + P`
- **Buscar en todo el proyecto** → `Ctrl + Shift + F` | `Cmd + Shift + F`
- **Saltar entre archivos abiertos** → `Ctrl + Tab` | `Cmd + Tab`

### 🖋️ Edición de Código

- **Duplicar línea actual** → `Shift + Alt + ↓/↑` | `Shift + Option + ↓/↑`
- **Mover líneas arriba/abajo** → `Alt + ↑/↓` | `Option + ↑/↓`
- **Eliminar línea completa** → `Ctrl + Shift + K` | `Cmd + Shift + K`
- **Seleccionar la misma palabra en varios lugares** → `Ctrl + D` | `Cmd + D`
- **Seleccionar todas las ocurrencias** → `Ctrl + Shift + L` | `Cmd + Shift + L`
- **Expandir selección palabra a palabra** → `Shift + Alt + →/←` | `Shift + Option + →/←`

### 🎭 Manejo de Pestañas y Paneles

- **Cerrar pestaña actual** → `Ctrl + W` | `Cmd + W`
- **Reabrir pestaña cerrada** → `Ctrl + Shift + T` | `Cmd + Shift + T`
- **Dividir editor en dos paneles** → `Ctrl + \` | `Cmd + \`
- **Moverse entre paneles** → `Ctrl + 1/2/3` | `Cmd + 1/2/3`

### 🏎️ Depuración y Terminal

- **Abrir terminal integrada** → ` Ctrl + \`` |  `Cmd + \``
- **Ejecutar código en terminal** → `Ctrl + Shift + B` | `Cmd + Shift + B`
- **Añadir punto de interrupción (debug)** → `F9`
- **Iniciar depuración** → `F5`
- **Parar depuración** → `Shift + F5`
- **Paso a paso en depuración** → `F10`

## ⚙️ Hacks y Configuraciones Avanzadas

### 1️⃣ **Modo Zen para Enfocarte al Máximo**

Si te distraes fácilmente, usa el **Modo Zen** para ocultar menús y concentrarte en el código:

```bash
Ctrl + K Z  # Windows/Linux
Cmd + K Z   # Mac
```

### 2️⃣ **Activar el Guardado Automático**

Evita perder cambios activando el guardado automático:

```json
"files.autoSave": "afterDelay"
```

### 3️⃣ **Mejorar la Experiencia con Extensiones Clave**

Aquí algunas extensiones que te harán la vida más fácil:

- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) ✨ → Autoformatea tu código.
- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) 🔍 → Mejora la visualización de Git.
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 🚀 → Recarga automática de proyectos web.
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) ⚠️ → Identifica errores en JavaScript y TypeScript.

### 4️⃣ **Crear Snippets Personalizados**

Si repites mucho un fragmento de código, crea un **snippet** propio:

1. Abre la paleta de comandos (`Ctrl + Shift + P` | `Cmd + Shift + P`)
2. Busca `Configure User Snippets`
3. Selecciona un archivo de lenguaje y añade:

```json
"Mi Snippet de Console": {
  "prefix": "log",
  "body": [
    "console.log('$1');",
    "$2"
  ],
  "description": "Atajo rápido para console.log"
}
```

Ahora, solo escribe `log` y presiona `Tab` para insertar `console.log()`. 🚀
